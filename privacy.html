Политика конфиденциальности и защиты данных

Последнее обновление: 1 января 2025 года

1
Общие положения

1.1. Настоящая Политика конфиденциальности (далее — "Политика") определяет порядок обработки и защиты информации о физических и юридических лицах (далее — "Пользователи"), использующих продукт "Синхро-Безопасность" (далее — "Продукт").

1.2. Целью Политики является обеспечение защиты прав и свобод человека и гражданина при обработке его персональных данных, в том числе защиты прав на неприкосновенность частной жизни, личную и семейную тайну.

1.3. Использование Продукта означает безоговорочное согласие Пользователя с настоящей Политикой и указанными в ней условиями обработки его персональных данных.

2
Обрабатываемые данные

2.1. При использовании демо-версии Продукта мы можем обрабатывать:

  • Имя и фамилия контактного лица
  • Название организации
  • Контактный телефон
  • Адрес электронной почты
  • Технические данные (IP-адрес, тип браузера, ОС)

2.2. При использовании Продукта в производственной среде:

  • Продукт работает локально на серверах Заказчика
  • Все данные сотрудников остаются внутри инфраструктуры Заказчика
  • Мы не имеем доступа к персональным данным сотрудников Заказчика
  • Обработка данных осуществляется силами Заказчика

2.3. Технические данные, которые обрабатывает Продукт:

  • Логины и хэши паролей (не в открытом виде)
  • Идентификаторы групп безопасности
  • Метаданные о структуре организации
  • Логи выполнения операций (без персональных данных)

3
Цели обработки данных

3.1. Обработка персональных данных Пользователей осуществляется в следующих целях:

  • Предоставление доступа к демо-версии Продукта
  • Оказание технической поддержки и консультаций
  • Информирование о новых версиях и обновлениях Продукта
  • Заключение и исполнение договорных обязательств
  • Соблюдение требований законодательства РФ

3.2. Продукт "Синхро-Безопасность" при эксплуатации Заказчиком осуществляет обработку данных в целях:

  • Автоматизации создания и управления группами безопасности
  • Синхронизации данных между 1С и системами контроля доступа
  • Обеспечения информационной безопасности предприятия
  • Аудита действий с учётными записями доступа

4
Принципы обработки данных

4.1. Обработка персональных данных осуществляется на основе следующих принципов:

  • Законность: Обработка осуществляется в соответствии с Федеральным законом №152-ФЗ "О персональных данных"
  • Ограничение цели: Данные обрабатываются только в заявленных целях
  • Минимизация: Обрабатываются только необходимые данные
  • Точность: Обеспечение актуальности обрабатываемых данных
  • Безопасность: Защита данных от несанкционированного доступа

4.2. Для Продукта "Синхро-Безопасность" действуют дополнительные принципы:

  • Локальность: Данные обрабатываются на серверах Заказчика
  • Прозрачность: Все операции логируются и могут быть проверены
  • Ответственность: Заказчик несёт ответственность за обработку данных своих сотрудников
  • Конфиденциальность: Технические данные не передаются третьим лицам

5
Меры безопасности

5.1. Для защиты персональных данных Пользователей мы применяем следующие меры безопасности:

  • Шифрование передаваемых данных по протоколу TLS 1.2 и выше
  • Регулярное обновление программного обеспечения и систем безопасности
  • Ограничение доступа к персональным данным сотрудниками
  • Резервное копирование и восстановление данных
  • Регулярный аудит систем безопасности

5.2. Продукт "Синхро-Безопасность" включает встроенные механизмы безопасности:

  • Хранение учётных данных в зашифрованном виде
  • Журналирование всех операций с доступом только для чтения
  • Валидация входных данных для предотвращения инъекций
  • Механизмы отката при ошибках синхронизации
  • Аудит изменений в группах безопасности

5.3. Рекомендации по безопасности для Заказчиков:

  • Установка Продукта в защищённом сегменте сети
  • Использование отдельных учётных записей для службы синхронизации
  • Регулярное обновление Продукта и зависимостей
  • Настройка правил брандмауэра для ограничения доступа
  • Мониторинг журналов выполнения операций

6
Права субъектов персональных данных

6.1. Субъект персональных данных имеет право:

  • На получение информации об обработке своих персональных данных
  • На доступ к своим персональным данным
  • На уточнение (обновление, изменение) своих персональных данных
  • На блокирование или уничтожение своих персональных данных
  • На отзыв согласия на обработку персональных данных
  • На обжалование действий или бездействия при обработке персональных данных

6.2. Для реализации своих прав субъект персональных данных может:

  • Направить запрос по электронной почте: vladimir_minkov@mail.ru
  • Обратиться по телефону: +7 (919) 157-79-00
  • Направить письменное обращение по почтовому адресу

6.3. Ответ на запрос будет предоставлен в течение 30 календарных дней с момента получения запроса.

7
Передача данных третьим лицам

7.1. Мы не передаем персональные данные Пользователей третьим лицам, за исключением:

  • Случаев, предусмотренных законодательством Российской Федерации
  • Получения явного согласия субъекта персональных данных
  • Необходимости исполнения договорных обязательств

7.2. Продукт "Синхро-Безопасность" при эксплуатации Заказчиком:

  • Не передаёт данные за пределы инфраструктуры Заказчика
  • Не осуществляет внешние вызовы без явного согласия Заказчика
  • Не собирает статистику использования без настроек Заказчика

7.3. В демо-версии Продукта данные могут использоваться для:

  • Анализа использования функций демо-версии
  • Улучшения пользовательского интерфейса
  • Отправки информационных материалов о Продукте

8
Хранение данных

8.1. Сроки хранения персональных данных:

  • Данные демо-пользователей: 1 год с момента последней активности
  • Данные клиентов: в течение срока действия договора + 5 лет
  • Технические логи: 6 месяцев
  • Финансовые документы: 5 лет

8.2. По истечении сроков хранения данные:

  • Аннулируются (уничтожаются) безопасным способом
  • Обеспечивается невозможность их восстановления
  • Составляется акт об уничтожении данных

8.3. Продукт "Синхро-Безопасность" предусматривает:

  • Настраиваемые сроки хранения журналов операций
  • Автоматическую ротацию лог-файлов
  • Возможность экспорта данных для архивирования
  • Механизмы очистки устаревших данных

9
Ответственность Заказчика

9.1. При использовании Продукта "Синхро-Безопасность" в производственной среде Заказчик обязан:

  • Обеспечить правовые основания для обработки персональных данных сотрудников
  • Назначить ответственного за обработку персональных данных
  • Уведомить Роскомнадзор об обработке персональных данных (при необходимости)
  • Обеспечить техническую защиту обрабатываемых данных
  • Ограничить доступ к данным в соответствии с должностными обязанностями

9.2. Заказчик самостоятельно определяет:

  • Какие данные сотрудников будут обрабатываться
  • Какие правила синхронизации будут применяться
  • Кто будет иметь доступ к настройкам Продукта
  • Как будут храниться резервные копии данных

9.3. Мы предоставляем:

  • Инструмент для автоматизации процессов управления доступом
  • Техническую документацию по настройке безопасности
  • Консультации по вопросам интеграции и безопасности
  • Обновления Продукта с исправлением уязвимостей

10
Заключительные положения

10.1. Настоящая Политика может изменяться в одностороннем порядке. Новая редакция Политики вступает в силу с момента её размещения на сайте.

10.2. Рекомендуется регулярно проверять актуальность Политики конфиденциальности.

10.3. Все предложения или вопросы по настоящей Политике следует направлять по контактам, указанным в разделе 6.2.

10.4. Действующая редакция Политики находится в свободном доступе на сайте Продукта.
