Главная / Возможности

Полный набор функций для промышленной безопасности

Синхро-Безопасность предлагает комплексное решение для автоматизации управления доступом в корпоративных средах промышленных предприятий

Интеграция с 1С

• Подключение к базам данных 1С
• Поддержка MSSQL и PostgreSQL
• Ежедневная автоматическая синхронизация
• Обновление изменённых данных
• Резервное копирование конфигураций

Работа с AD/Samba

• Поддержка Active Directory Windows
• Совместимость с Samba DC (RedOS)
• Автоматическое создание групп
• Управление организационными единицами
• Наследование политик безопасности

Управление пользователями

• Автоматическое добавление/удаление
• Перемещение между группами
• Назначение минимальных прав
• Блокировка устаревших доступов
• Управление исключениями

Безопасность

• Назначение необходимых прав
• Шифрование передаваемых данных
• Двухфакторная аутентификация
• Аудит всех изменений

Производительность

Отчётность
