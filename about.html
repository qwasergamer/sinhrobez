Синхро-Безопасность - промышленная автоматизация безопасности

Суть решения

Синхро-Безопасность - это специализированный программный модуль для автоматической синхронизации организационной структуры предприятия из системы 1С с группами безопасности в Active Directory (Windows) и Samba DC (RedOS). Решение разработано специально для нужд крупных промышленных предприятий.

Как это работает

1. Модуль получает актуальную оргструктуру из базы данных 1С
2. Сравнивает её с существующими группами безопасности
3. Автоматически создаёт недостающие группы
4. Назначает пользователей в соответствии с должностями
5. Управляет правами доступа
6. Ведёт детальный журнал изменений

Для промышленных предприятий

• Нефтегазовые компании
• Промышленные комбинаты
• Крупные производства
• Энергетические компании
• Промышленные холдинги
• Предприятия с требованиями информационной безопасности

Преимущества

• Снижение рисков безопасности на 90%
• Экономия времени администраторов на 70%
• Исключение человеческих ошибок
• Соответствие требованиям стандартов
• Быстрая адаптация к изменениям структуры

Технический процесс работы

1. Сбор данных

Ежедневное получение оргструктуры из 1С

2. Анализ

Сравнение с текущими группами AD/Samba

3. Синхронизация

Создание/обновление групп безопасности

4. Назначение прав

Автоматическое распределение доступов

5. Отчётность

Формирование детальных отчётов
