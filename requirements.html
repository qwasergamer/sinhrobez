Требования к инфраструктуре

Active Directory

• Windows Server 2012 R2 или новее
• Функциональный уровень домена 2012 R2+
• Доступ к контроллеру домена
• Права на чтение оргструктуры
• Права на управление группами

Samba DC

• Samba 4.7 или новее
• RedOS 7.3+ или Альт Linux 10+
• Доступ по протоколу LDAP
• Учётная запись с правами администратора
• Настроенные политики безопасности

1С:Предприятие

• Версия платформы 8.3.10 или новее
• Доступ к базе данных 1С
• Учётная запись с правами чтения
• Разрешение на доступ к оргструктуре
• Стабильное сетевое соединение

Требования к правам доступа

Учётная запись

  • Доменная учётная запись
  • Права локального администратора на сервере
  • Членство в группе Domain Admins (рекомендуется)
  • Или делегированные права на OU
  • Права на создание/удаление групп безопасности

Безопасность

  • Доступ по защищённому каналу (LDAPS)
  • Шифрование передаваемых данных
  • Ограничение доступа к серверу модуля
  • Регулярное обновление сертификатов
  • Аудит действий учётной записи

Доступ к данным

  • Чтение данных оргструктуры из 1С
  • Доступ только к необходимым таблицам
  • Ограничение времени доступа
  • Журналирование всех запросов
  • Резервное копирование конфигураций

Производительность

500-1000 пользователей

Сервер: 4 ядра, 8 GB RAM
Время синхронизации: 2-5 минут

1000-5000 пользователей

Сервер: 8 ядер, 16 GB RAM
Время синхронизации: 5-10 минут

5000-10000 пользователей

Сервер: 12+ ядер, 32 GB RAM
Время синхронизации: 10-15 минут

* Время синхронизации указано для типовой оргструктуры с 3-4 уровнями вложенности
